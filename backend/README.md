# Time Tracker — Django backend

This folder is the **Django API**. Run all backend commands from here (e.g. `pip install -r requirements.txt`, `python manage.py migrate`, `python manage.py runserver`).

- **Vercel:** In your Vercel project settings, set **Root Directory** to `backend` so the build uses this folder.
- **Railway / Render:** Set the project root or "Root Directory" to `backend` when connecting the repo.

See the main [README](../README.md) in the project root for overview and API details.
